⚙️ How to configure your mail server
For exceweyn.run.place (or any custom domain), you need to set up email routing so that every email arriving at *@yourdomain.com is forwarded as an HTTP POST to the webhook URL above.

Option A — Cloudflare Email Routing (free & easy):
1. Go to Cloudflare → your domain → Email → Email Routing
2. Enable Email Routing, add a catch-all rule
3. Set action = Send to a Worker
4. Create a Worker that does fetch(webhookUrl, {method:'POST', body: emailJson})

Option B — Mailgun (easiest webhook support):
1. Add your domain in Mailgun → Receiving → Routes
2. Create route: match all → Forward → paste your webhook URL above
3. Set MX records on your domain to Mailgun's servers

Option C — Forwardemail.net (free):
1. Add domain → create catch-all webhook
2. Paste the webhook URL above as the destination
3. Set their MX records on your domain DNS
